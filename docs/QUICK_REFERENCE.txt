# Quick Reference - Standard Prescription Templates

## 🎯 What Was Built

A complete prescription template system allowing doctors to:
- Save favorite prescriptions as reusable templates
- Search templates by keyword while creating prescriptions  
- Load templates with one click
- Edit template fields per patient
- Manage templates (CRUD) in dashboard

## 🔄 User Workflow

### 1️⃣ Create Template
```
Dashboard → Prescription Templates → Create New → Fill Form → Save
```

### 2️⃣ Add Items to Template
```
Template View → Add Medicine Form → Submit (AJAX)
                → Add Test Form → Submit (AJAX)
```

### 3️⃣ Use Template in Prescription
```
Create Prescription → Search Template in "Quick Load" → Load → Edit → Save
```

### 4️⃣ Save Prescription as Template
```
Create/Edit Prescription → Save as Template → Fill Modal → Save
```

## 🎓 For Developers

### Adding New Template Fields
1. Add to model: `hospital/models.py`
2. Add to form: `hospital/forms.py`
3. Update HTML: Template files
4. Create migration: `python manage.py makemigrations`

### Customizing Search
Edit in `views.py` search_prescription_templates():
```python
templates.filter(
    Q(name__icontains=query) | 
    Q(keyword__icontains=query) |
    Q(description__icontains=query)
)
```

### Changing Limits
In `search_prescription_templates()`:
```python
templates[:10]  # Change limit here
```

## ❌ Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Templates not showing | Check `is_active=True`, verify search query |
| AJAX errors | Check console, verify clinic slug in URLs |
| Permission denied | Verify user role is 'doctor' |
| Items not loading | Check template has medicines/tests |
